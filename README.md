# medicure
Medicure – Responsive Medical Website A modern, sleek, and responsive medical service platform designed for healthcare accessibility.

Features
Modern UI/UX: Clean, dark-themed design optimized for readability.

Responsive Layout: Fully adaptive for mobile, tablet, and desktop using Tailwind CSS.

Service Showcase: Dedicated sections for various medical departments (Cardiology, Neurology, Dermatology, etc.).

Interactive Elements: Smooth animations and intuitive navigation.

Optimized Performance: Fast-loading, clean code structure.

🛠 Tech Stack
Frontend: HTML5, CSS3, Tailwind CSS

Interactivity: JavaScript (Vanilla)

Animations: AOS (Animate On Scroll) Library
